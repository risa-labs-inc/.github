## Our Products

### BOSS - Business Operating System Service

A multiplatform desktop application combining intelligent automation, RPA capabilities, and comprehensive development tools.

- **Integrated Browser** with WebAuthn/Passkey authentication
- **Terminal Integration** for seamless CLI workflows
- **Workspace Management** with customizable layouts
- **AI-Powered Automation** for repetitive tasks

[![Download BOSS](https://img.shields.io/badge/Download-BOSS-blue?style=for-the-badge)](https://github.com/risa-labs-inc/BOSS-Releases/releases)
[![Homebrew](https://img.shields.io/badge/brew-install%20boss-orange?style=for-the-badge)](https://github.com/risa-labs-inc/homebrew-boss)

```bash
# Install via Homebrew (macOS)
brew tap risa-labs-inc/boss
brew install --cask boss
```

### BossTerm - Pure Kotlin Terminal Emulator

A high-performance terminal emulator built in pure Kotlin with SSH and PTY support. Powers the terminal integration in BOSS.

[![Download BossTerm](https://img.shields.io/badge/Download-BossTerm-green?style=for-the-badge)](https://github.com/kshivang/BossTerm/releases)
